# 🍎 Intelligent Food Management System

A full-stack MERN-based application to manage food inventory, track expiry dates, and send smart notifications.

## 🚀 Features
- User Authentication (Login/Register/Forgot Password)
- Add & Manage Products
- Expiry Tracking (Active / Expiring Soon / Expired)
- Daily Notifications for expiring items
- Stock Management (Sell system)
- Profile Management

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB Atlas

## 📌 Highlights
- Smart expiry logic
- Real-time dashboard updates
- One notification per product per day
- Secure password handling using bcrypt
